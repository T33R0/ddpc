
--- TABLE: maintenance_log ---
id: string (uuid) - Primary Key.

Note:
This is a Primary Key.<pk/>
user_vehicle_id: string (uuid) - Foreign Key to user_vehicle.

Note:
This is a Foreign Key to `user_vehicle.id`.<fk table='user_vehicle' column='id'/>
service_interval_id: string (uuid) - Foreign Key to service_intervals (links to a scheduled interval, if applicable).

Note:
This is a Foreign Key to `service_intervals.id`.<fk table='service_intervals' column='id'/>
cost: number (numeric) - Total cost of the service.
odometer: integer (integer) - Mileage at time of service.
event_date: string (timestamp with time zone) - Date of the service (or planned date).
created_at: string (timestamp with time zone) - Record creation timestamp.
notes: string (text) - User notes.
service_provider: string (text) - Who performed the service.
service_item_id: string (uuid) - Foreign Key to service_items (classifies the service).

Note:
This is a Foreign Key to `service_items.id`.<fk table='service_items' column='id'/>
status: string (text) - Status of the record ('Plan', 'History', 'Archive').

--- TABLE: service_intervals ---
id: string (uuid) - Primary Key.

Note:
This is a Primary Key.<pk/>
user_id: string (uuid) - The owner of the vehicle referencing the service_intervals
name: string (text) - Name of the scheduled service.
interval_months: integer (integer) - Time interval in months.
interval_miles: integer (integer) - Mileage interval.
description: string (text) - Description of the service.
user_vehicle_id: string (uuid) - Foreign Key to user_vehicle.

Note:
This is a Foreign Key to `user_vehicle.id`.<fk table='user_vehicle' column='id'/>
due_date: string (date) - The calculated due date for the *next* service.
due_miles: integer (integer) - The calculated due mileage for the *next* service.

Note:

--- TABLE: service_items ---
id: string (uuid) - Primary Key.

Note:
This is a Primary Key.<pk/>
category_id: string (uuid) - Foreign Key to service_categories.

Note:
This is a Foreign Key to `service_categories.id`.<fk table='service_categories' column='id'/>
name: string (text) - Name of the service item.
description: string (text) - Default description.

--- TABLE: job_plans ---
id: string (uuid) - Primary key.

Note:
This is a Primary Key.<pk/>
user_id: string (uuid) - ID of the user creating the plan.
maintenance_log_id: string (uuid) - Foreign Key to maintenance_log (links the plan to a vehicle and service item).

Note:
This is a Foreign Key to `maintenance_log.id`.<fk table='maintenance_log' column='id'/>
mod_log_id: string (uuid) - Foreign Key to mod_log_id

Note:
This is a Foreign Key to `mods.id`.<fk table='mods' column='id'/>
name: string (text) - Title of the job plan.
created_at: string (timestamp with time zone) - Record creation timestamp.

--- TABLE: job_steps ---
id: string (uuid) - Primary Key.

Note:
This is a Primary Key.<pk/>
job_plan_id: string (uuid) - Foreign Key to job_plans.

Note:
This is a Foreign Key to `job_plans.id`.<fk table='job_plans' column='id'/>
step_order: integer (integer) - Sequence number of the step.
description: string (text) - Main instruction for the step.
notes: string (text) - Additional details or warnings.
is_completed: boolean (boolean) - Completion status.
created_at: string (timestamp with time zone) - Record creation timestamp.
is_completed_reassembly: boolean (boolean) - Completion status for reassembly phase.
